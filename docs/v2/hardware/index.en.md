# Hardware Reference

This section summarizes the hardware properties of Robutek.

## General hardware concepts

The Robutek hardware mainly uses these peripherals:

- `GPIO` for digital inputs and outputs
- `ADC` for reading analog values from the reflectance sensors
- `PWM` for servo and motor control
- `Quadrature encoders` for motor movement feedback
- `I2C` for sensors
- `WS2812B` for RGB lighting
